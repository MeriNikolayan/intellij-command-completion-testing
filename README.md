Command Completion Exploratory Testing

This repository contains the results and supporting files for exploratory testing of the Command Completion feature in IntelliJ IDEA. 
The Command Completion functionality helps developers write and transform code by suggesting context-aware IDE actions directly in the editor.

Contents

1. Source Code: Sample Java project used for testing Command Completion scenarios.
2. TestReport.pdf: Detailed documentation of test cases, statuses, suggestions, and testing results.
3. Build Files: Gradle build scripts and wrapper files for easy project setup.

Purpose

The project demonstrates common mechanisms and usability aspects of Command Completion, including code generation, refactoring, quick-fixes, high-level transformations, and navigation actions.

How to Use

1. Clone the repository: git clone https://github.com/MeriNikolayan/intellij-command-completion-testing.git
2. Open the project in IntelliJ IDEA.
3. Review the source code and documentation.

Notes

The test report provides suggestions for further testing and usability improvements.
